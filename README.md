## Задание 1. Текстовые файлы

### Текст задачи 
Найти разность максимального и минимального элементов.

### Алгоритм решения

По результатам обработки вычисляется разность между максимальным и минимальным значениями. Полученный результат выводится на экран.

## Задание 10. Dictionary/SortedList

### Текст задачи 

На вход программы подаются сведения о результатах соревнований по школьному многоборью.
Многоборье состоит из соревнований по четырем видам спорта, участие в каждом из которых
оценивается баллами от 0 до 10 (0 баллов получает ученик, не принимавший участия в
соревнованиях по данному виду спорта). Победители определяются по наибольшей сумме
набранных баллов. Известно, что общее количество участников соревнований не превосходит
100.

### Алгоритм решения

Из текстового файла считываются сведения об участниках школьного многоборья и набранных ими баллах по четырем видам спорта. Для каждого участника вычисляется суммарный результат.

После обработки всех данных определяются участники, занявшие призовые места. Если несколько участников имеют результат, равный результату третьего места, они также включаются в итоговый список.
